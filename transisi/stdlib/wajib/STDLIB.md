# Standard Library (Wajib)

Berikut adalah daftar modul yang tersedia di pustaka standar "wajib" Morph.

- [`berkas`](./BERKAS.md): Operasi sistem file.
- [`daftar`](./DAFTAR.md): Operasi pada daftar (list).
- [`matematik`](./matematik.fox): Fungsi dan konstanta matematika.
- [`teks`](./TEKS.md): Operasi pada string (teks).
- [`waktu`](./WAKTU.md): Operasi pada objek tanggal dan waktu.
