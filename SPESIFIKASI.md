# Spesifikasi Bahasa Pemrograman MORPH

Dokumen ini berisi spesifikasi resmi untuk bahasa pemrograman MORPH. Tujuannya adalah untuk mendefinisikan aturan sintaks (bagaimana kode ditulis) dan semantik (apa arti dari kode tersebut).

## Rencana Pengembangan Awal

- [ ] Spesifikasi Tipe Data Dasar
- [ ] Spesifikasi Komentar
- [ ] Spesifikasi Variabel

## Bab 1: Tipe Data Dasar

Tipe data dasar adalah blok bangunan paling fundamental dalam bahasa MORPH.

### 1.1 Bilangan Bulat (Integer)

- [ ] **Sintaks**: Bilangan bulat ditulis sebagai urutan satu atau lebih digit desimal (0-9).
  - *Contoh*: `1`, `42`, `100`, `0`
- [ ] **Semantik**: Merepresentasikan nilai numerik utuh tanpa komponen pecahan.

### 1.2 Bilangan Desimal (Float)

- [ ] **Sintaks**: Bilangan desimal ditulis dengan bagian bulat dan bagian pecahan, yang dipisahkan oleh tanda titik (`.`).
  - *Contoh*: `3.14`, `0.5`, `99.9`
- [ ] **Semantik**: Merepresentasikan nilai numerik yang mungkin memiliki komponen pecahan.

### 1.3 Boolean

- [ ] **Sintaks**: Tipe data boolean hanya memiliki dua nilai literal:
  - `benar` (merepresentasikan *true*)
  - `salah` (merepresentasikan *false*)
- [ ] **Semantik**: Merepresentasikan nilai kebenaran logika, yang biasanya digunakan dalam operasi kondisional.
